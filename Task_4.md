## Решение
В рамках решения данного задания будет использоваться поисковая система Яндекс, так как задание явно связано с российским сегментом интернета.

В процессе CTF для данного таска были представлены две подсказки:
![hint1](/imgs/Pasted%20image%2020250720120609.png)
![hint2](/imgs/Pasted%20image%2020250721103139.png)

Таким образом у нас есть зацепка о названии трека и подсказка, что искать аккаунт следует во Вконтакте.

Попробуем найти трек.  
По запросу `популярные песни дуэта "100 gecs"` получаем, что искали:

![Задача](/imgs/Pasted%20image%2020250720115945.png)

Название песни - `Money Machine`  
Найдём пользователей Вконтакте с таким именем

<img src="https://github.com/AronHopeless/CyberCamp2025_guide/blob/main/imgs/Pasted%20image%2020250720152953.png" width="500">

Последовательно проверим каждого пользователя, их подписки, сообщества и посты (если есть).  
Подозрительным в этом отношении является пользователь [@id894819633](https://vk.com/id894819633).  
Среди его сообществ присутствует *зоомагазин* с названием "secret place" [link](https://vk.com/club231398698?from=groups).  
Пользователь является единственным участником и создателем сообщества.  
Дата создания сообщества - незадолго до проведения CTF.

<img src="https://github.com/AronHopeless/CyberCamp2025_guide/blob/main/imgs/Pasted%20image%2020250721102255.png" width="500">

В сообществе создан единственный пост, содержащий три слова (вспоминаем про три слова-подсказки из задания):
```
кора
пони
поиск
```

<img src="https://github.com/AronHopeless/CyberCamp2025_guide/blob/main/imgs/Pasted%20image%2020250721102439.png" width="500">

*\*пропущенная логическая связь*\*

Имея в распоряжении три *магических* слова, можно обратиться к ресурсу [what3words](https://what3words.com) для поиска географического расположения
Впишем их в поисковую строку, система предложит вариант с исправленным синтаксисом: `///кора.пони.поиск` 

![Задача](/imgs/Pasted%20image%2020250721174720.png)

На данной геоклетке имеем здание по адресу `Москва, 1-й Боткинский пр-д, 7 стр. 1`

<img src="https://github.com/AronHopeless/CyberCamp2025_guide/blob/main/imgs/Pasted%20image%2020250721174859.png" width="750">

Это искомый адрес и вот он в формате третьего флага:
```
Москва, 1-й Боткинский проезд, дом 7с1
```
